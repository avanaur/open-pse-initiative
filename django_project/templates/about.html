About

OpenPSE is an open-source project which has the following goals:

  • Store end-of-day quotes from the Philippine Stock Exchange(PSE)
  • Provide a public REST API for the data
  • Provide downloadable end-of-day quotes files(CSV, JSON, XML)
  • Aggregate news from different sources regarding companies listed in the PSE
  • Provide users with stock charts and indicators

The source code may be found on Github.
